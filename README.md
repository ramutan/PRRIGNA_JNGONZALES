## 📝 Game Description
**Pixel Dash is a high-octane, retro-style endless runner developed as a final requirement for a Programming 2 course at the University of Perpetual Help System DALTA. Built using pure Java Swing and AWT, the game challenges players to navigate a series of increasingly difficult obstacles to achieve the highest score possible. This version introduces dynamic features such as animated flying birds, a smooth day and night cycle, and strategic power-ups like shields and coin magnets. Players can personalize their experience through character customization and a name input system, while a persistent leaderboard tracks the top ten rankings locally. By integrating complex mechanics like double-jumping and sliding with robust state management, the project serves as a comprehensive demonstration of game logic and collision detection within a single-file implementation.

The goal is simple: **Don't stop, don't hit anything, and keep dashing.**

## 🕹️ Controls & Mechanics
Mastering the movement is the only way to survive the dash. You can use either the Arrow keys or the classic WASD setup.
| Key | Action |
|---|---|
| ↑ / W / Space | Jump (double-jump in air) |
| ↓ / S | Slide (hold) |
| P | Pause / resume |
| Esc | Back to main menu |
| Enter | Confirm in menus / name input |
| ←/→ | Change customization option |
| Backspace | Delete letter (name input) |
| C | Clear leaderboard (on leaderboard screen) |

### **Basic Movement**
*   **Jump:** Press `UP Arrow`, `W`, or `Space Bar` to leap over ground obstacles and gaps.
*   **Crouch/Slide:** Press `DOWN Arrow` or `S` to duck under low-hanging hazards.

### **Objective**
*   **Avoid Obstacles:** Dodge, jump, or slide past incoming barriers.
*   **High Score:** The longer you survive, the faster the game becomes, increasing the challenge and your potential score.
